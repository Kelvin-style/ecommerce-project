# E-Commerce Website

## Project Overview
This is a PHP and MySQL based e-commerce website developed as a practical web development project. The system allows administrators to manage products while customers can browse products, add items to the shopping cart, and place orders.

## Features

- Product Management
- Shopping Cart
- User Registration and Login
- Checkout System
- Order Management
- Responsive Design

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
- XAMPP

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Kelvin_style/ecommerce-project.git
```

2. Copy the project into:

```
C:\xampp\htdocs\
```

3. Start Apache and MySQL using XAMPP.

4. Import the database:

```
ecommerce_db.sql
```

5. Open your browser:

```
http://localhost:8080/ecommerce/
```
